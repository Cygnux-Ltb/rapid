时间格式化

时间戳格式化

/**
 * 格式化时间戳
 *     timestamp ：时间戳，支持秒和毫秒
 *     fmt       : 格式化。 例如：yyyy-MM-dd HH:mm:ss
 */
var timeText = javaex.dateFormat(timestamp, fmt);

修改时间显示（返回：N天前、N小时前...）

/* 
 * 修改时间显示
 * time : 传入的时间，格式yyyy-MM-dd HH:mm:ss
 */
var timeText = javaex.changeTimeText(time);

修改时间显示（会自动修改为：N天前、N小时前...）
