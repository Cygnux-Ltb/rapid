本地数据存储

cookie

本地保存的数据会发送给服务器

方法 说明 
javaex.setCookie(key, value, time);
 设置cookie。time表示存储多少天（单位：天），若缺省表示关闭浏览器后，cookie消失 
var str = javaex.getCookie(key);
 获取指定cookie的值 
javaex.deleteCookie(key);
 删除指定的cookie

sessionStorage

生命周期为关闭浏览器窗口

在同一个页面下，数据可以共享

方法 说明 
javaex.setSessionStorage(key, value);
 设置sessionStorage 
var str = javaex.getSessionStorage(key);
 获取指定sessionStorage的值 
javaex.deleteSessionStorage(key);
 删除指定的sessionStorage；key为空时，清除所有

localStorage

生命周期为永久生效，除非手动删除，否则页面关闭也会存在

可以多页面共享（同一浏览器可以共享）

方法 说明 
javaex.setLocalStorage(key, value);
 设置localStorage 
var str = javaex.getLocalStorage(key);
 获取指定localStorage的值 
javaex.deleteLocalStorage(key);
 删除指定的localStorage；key为空时，清除所有
