<script>
	javaex.classify({
		selector : "ul.javaex-classify-ul",
		// 点击后，返回一个回调函数
		callback: function (rtn) {
			console.log(rtn);
			// 数组转为字符串
			var str = rtn.join("&");
			console.log(str);
		}
	});
</script>

参数说明

参数 默认值 说明 版本
selector jquery选择器，指定到ul即可
isInit false 是否初始化就自动调用回调函数
callback 返回一个数组形式，包含了分类的id和对应的类型的值。分类和对应的值中间有等号 ["leixing=", "fengge=6"]
