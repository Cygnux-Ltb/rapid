DeleteDialog 删除询问框

删除专用弹出层，常用于列表删除数据

市区 省区 国别 操作
苏州 江苏省 中国 删除 
<table class="javaex-table">
	<thead>
		<tr>
			<th>市区</th>
			<th>省区</th>
			<th>国别</th>
			<th>操作</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>苏州</td>
			<td>江苏省</td>
			<td>中国</td>
			<td><a href="javascript:;" style="color:#fa2c19;" onclick="doDelete(this, 'xxxx')">删除</a></td>
		</tr>
	</tbody>
</table>

<script>
	// obj为当前点击的元素对象
	function doDelete(obj, id) {
		// obj是必须的
		javaex.deleteDialog(obj, {
			content : "确定要删除么？",
			confirm : function() {
				console.log("获取变量值：" + id);
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	}
</script>

参数说明

参数 默认值 说明 版本
content 弹出层内容
confirmName 确定 确定按钮名称
cancelName 取消 取消按钮名称
offsetTop 0 上偏移量
confirm 点击确定按钮后的回调函数
cancel 点击取消按钮后的回调函数
opend 打开弹窗后的回调函数
opend : function() {
	console.log("弹窗已打开");
}
