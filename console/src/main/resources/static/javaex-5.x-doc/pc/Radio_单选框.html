Radio 单选框

基础用法

  • 默认点击文字也会勾选，可以不用在外面包一层label。
  • 如果没有文字，则必须写一个空格 或者 直接用label标签包起来。
  • 如果是ajax获取数据后拼接的代码，需要调用 javaex.render(); 重新渲染样式。

伪单选框

一定要在外面包一层label，并且需要自己写一点点样式

<style>
.javaex-btn.blue {
    color: rgb(255, 255, 255);
    background-color: rgb(64, 158, 255);
}
.javaex-btn.wathet {
    color: rgb(16, 140, 238);
    background-color: rgb(234, 246, 254);
}
</style>

<label class="javaex-btn"><input type="radio" fake name="radio3" value="1" checked/>入门型</label>
<label class="javaex-btn"><input type="radio" fake name="radio3" value="2"/>标准型</label>
<label class="javaex-btn"><input type="radio" fake name="radio3" value="3"/>企业型</label>
<label class="javaex-btn"><input type="radio" fake name="radio3" value="4"/>豪华型</label>
 
<script>
	javaex.fakeRadio({
		name : "radio3",
		uncheckClass : "wathet", // 未选中时添加的class
		checkedClass : "blue", 	// 选中时添加的class
		isInit : true,			// 初始化是否调用回调函数
		callback : function(rtn) {
			console.log(rtn.val);
			console.log(rtn.text);
		}
	});
	
	// 支持jquery原生方法取值
	$(':radio[name="radio3"]').change(function() {
		console.log($(this).val());
	});
</script>

参数说明

参数 默认值 说明 版本
name radio的name属性
uncheckClass 未选中时往label标签添加的class
checkedClass 选中时往label标签添加的class
isInit false 初始化是否自动调用回调函数
callback 返回选中的值和文本
