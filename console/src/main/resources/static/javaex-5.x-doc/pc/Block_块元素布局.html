参数说明

属性 说明
class="javaex-block" 该属性修饰的div表示一块内容区域
追加 has-border 属性可以追加边框。 写法：class="javaex-block has-border"
class="javaex-banner" 该属性修饰的div表示块元素的名称，是专门用来修饰class="javaex-block"的，写在class="javaex-block"内。可以缺省
class="javaex-block-fixed" 添加该代码，可以在标题前多加一根竖线
class="javaex-main" 或 class="javaex-main-10" 该属性修饰的div表示填写正文内容，写在class="javaex-block"内，与class="javaex-banner"同级
该属性的div会产生20px的内边距
class="javaex-main-10"会产生10px的内边距
