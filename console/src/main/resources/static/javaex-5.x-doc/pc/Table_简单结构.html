class="javaex-table"可以附加如下属性

属性 说明
color1、color2 color1表示奇数行着色、color2表示偶数行着色，例如：class="javaex-table color2"
unhover 默认鼠标滑过时，当前行会变色。class加上 unhover 可以不变色
th-c、td-c 可以让内容居中
th-c-1、td-c-1、th-c-2、td-c-2 可以让内容居中（只对第1列和第2列有效）
no-vertical-line 可以去除竖线
no-cross-line 可以去除横线
no-border-line 可以去边框线
no-line 可以去除竖线、横线、边框线
vertical-line-white 可以使边框外的竖线变为白色
