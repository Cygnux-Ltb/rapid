Input 输入框

写在最前面

  • 如果你不想input获取焦点时，添加蓝色边框，添加 class="original" 即可，这是为了防止和前台搜索框的input产生冲突
  • 如果input是只读的，可以添加 class="readonly" 来添加样式

输入框（默认）

<input type="text" class="javaex-text" placeholder="提示信息" autocomplete="off" />

输入框（获取焦点时，无边框）

<input type="text" class="javaex-text original" placeholder="提示信息" />

输入框（只读）

<input type="text" class="javaex-text readonly" value="我是只读的" readonly autocomplete="off" />

搜索框

<span class="javaex-search-group">
	<input type="text" class="javaex-text" placeholder="提示信息" />
	<button class="javaex-btn blue">搜索</button>
</span>
