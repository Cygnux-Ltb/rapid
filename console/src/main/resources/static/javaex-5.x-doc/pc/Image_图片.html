Image 图片

基础用法

基础用法与原生 img 标签一致，可以设置 src、width、height、alt 等原生属性。

<div class="javaex-image" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" />
</div>

填充模式

通过 fit 属性可以设置图片填充模式，等同于原生的 object-fit 属性，可选值见下方表格。

  • contain
  • cover
  • fill
  • none
  • scale-down
// contain
<div class="javaex-image" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" fit="contain" />
</div>

// cover
<div class="javaex-image" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" fit="cover" />
</div>

// fill
<div class="javaex-image" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" fit="fill" />
</div>

// none
<div class="javaex-image" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" fit="none" />
</div>

// scale-down
<div class="javaex-image" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" fit="scale-down" />
</div>

圆形图片

通过 round 属性可以设置图片变圆，注意当图片宽高不相等且 fit 为 contain 或 scale-down 时，将无法填充一个完整的圆形。

<div class="javaex-image javaex-image-round" style="width: 80px; height: 80px;">
<img class="javaex-img" src="http://webimg.javaex.cn/Fmxl5u5ybgZWUMxLtf81ZYWPP_xL" fit="cover" />
</div>
