TextArea 文本域

默认模式（限制高度）

<textarea class="javaex-desc" placeholder="请填写简介"></textarea>

高度自适应

<textarea id="test1" class="javaex-desc" placeholder="请填写简介"></textarea>

<script>
	javaex.textarea({
		id : "test1",
		autoHeight : true
	});
</script>

显示字数统计

0/50
<textarea id="test2" class="javaex-desc" placeholder="请填写简介"></textarea>
<div class="javaex-field-word-limit"><span id="checklen">0</span>/50</div>

<script>
	javaex.textarea({
		id : "test2",
		autoHeight : true,
		checklenId : "checklen",
		maxlength : 50,
		callback : function() {
			console.log("已达到最大长度");
		}
	});
</script>

参数说明

参数 默认值 说明 版本
id textarea的ID 5.3.x
checklenId 数字统计的ID 5.3.x
autoHeight false 是否自适应高度 5.3.x
maxlength 最大输入字数，为0时表示不限制 5.3.x
callback 打开弹窗后的回调函数
callback : function() {
	console.log("已达到最大长度");
}
 5.3.x
