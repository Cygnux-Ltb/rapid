CircleProgress 环形进度条

基础用法

<div id="progress1" class="javaex-circleprogress" style="height: 100px;width: 100px;"></div>

<script>
	javaex.progress({
		id : "progress1",
		mode : "circle",
		percent : 50,
		isShowPercent : true
	});
</script>

参数说明

参数 默认值 说明 版本
id 指定操作哪个进度条
mode 环形进度条必须填写circle
width 5 进度条的宽度
percent 0 进度条百分比，例如：40 或 "40"
isShowPercent false 是否显示百分比文字（如果要显示百分比，则需要自己为 class="progress" 设定 line-height）
