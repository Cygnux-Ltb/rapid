Message 消息提示-适用于门户

<button id="submit" class="javaex-btn primary">提交中提示</button>
<button id="success" class="javaex-btn success">成功提示</button>
<button id="error" class="javaex-btn danger">异常提示</button>

<script>
	$("#submit").click(function() {
		javaex.message({
			content : "数据提交中，请稍候...",
			type : "submit"
		});
	});
	
	$("#success").click(function() {
		javaex.message({
			content : "操作成功",
			type : "success"
		});
	});
	
	$("#error").click(function() {
		javaex.message({
			content : "操作失败",
			type : "error"
		});
	});
</script>

参数说明

参数 默认值 说明 版本
hasMask true hasMask==true时，会有遮罩层，将无法再点击页面其他元素（防止用户重复提交）。
type=="submit"时，遮罩层是黑色半透明的；type=="success" 或 "error" 时，遮罩层是透明的
content 弹出层内容
type success submit:提交中。 success:成功。 error:失败
live 2000 提示存在时间，单位:毫秒 （仅在type为 success 或 error 时有效）
timeout 0 提交超时时间（单位:毫秒），填0时，不会有超时提示
timeoutText 网络请求超时，请稍后重试 超时提示
opend 打开后的回调函数
opend : function() {
	console.log("弹窗已打开");
}
