Tag 标签

写在最前面

  • 该组件没有用回调函数返回标签数组，而是采用input来存储数据，数据格式为： xxx,xxxx
  • 是为了迎合 ajax 序列化提交

基础用法

最多添加6个标签

<div id="tag" class="javaex-tagbox"></div>
<input type="hidden" id="tag-result" name="tag" value="" />
<p class="javaex-hint">最多添加6个标签</p>

<script>
	javaex.tag({
		id : "tag",
		inputId : "tag-result",
		tags : ['回显数据1', '回显数据2']
	});
</script>

参数说明

参数 默认值 说明 版本
id javaex-tagbox的ID
inputId 记录最终结果的input的id。结果格式为 xxx,xxxx
maxNum 6 标签数量上限
tags 标签回显，数组或list
