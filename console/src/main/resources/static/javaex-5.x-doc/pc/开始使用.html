开始使用

首先，创建一个HTML5的网页文档

<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>无标题文档</title>
</head>

<body>
</body>
</html>

然后，你需要加载以下文件

<!--字体图标-->
<link href="javaex-5.5.1/pc/css/icomoon.css" rel="stylesheet" />
<!--骨架样式-->
<link href="javaex-5.5.1/pc/css/common.css" rel="stylesheet" />
<!--皮肤-->
<link href="javaex-5.5.1/pc/css/skin/default.css" rel="stylesheet" />
<!--jquery，如果不使用jquery高版本的新功能的话，不建议修改版本-->
<script src="javaex-5.5.1/pc/lib/jquery-1.7.2.min.js"></script>
<!--核心组件，不要压缩，压缩后部分功能可能会失效-->
<script src="javaex-5.5.1/pc/js/javaex.js"></script>
<!--表单验证-->
<script src="javaex-5.5.1/pc/js/javaex-formVerify.js"></script>

如果你只是想测试，可以加载远程文件进行测试

<!--字体图标-->
<link href="https://javaex.javaex.cn/javaex-5.5.1/pc/css/icomoon.css" rel="stylesheet" />
<!--骨架样式-->
<link href="https://javaex.javaex.cn/javaex-5.5.1/pc/css/common.css" rel="stylesheet" />
<!--皮肤-->
<link href="https://javaex.javaex.cn/javaex-5.5.1/pc/css/skin/default.css" rel="stylesheet" />
<!--jquery-->
<script src="https://javaex.javaex.cn/javaex-5.5.1/pc/lib/jquery-1.7.2.min.js"></script>
<!--核心组件-->
<script src="https://javaex.javaex.cn/javaex-5.5.1/pc/js/javaex.js"></script>
<!--表单验证-->
<script src="https://javaex.javaex.cn/javaex-5.5.1/pc/js/javaex-formVerify.js"></script>
