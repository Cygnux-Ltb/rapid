Dialog 对话框

可以最大化、最小化，适用于嵌入ifram表单

<button id="demo1" class="javaex-btn primary">点击测试</button>

<script>
	$("#demo1").click(function() {
		javaex.dialog({
			type : "window",
			url : "http://www.javaex.cn/",	// 页面地址或网址或请求地址
			width : "600",	// 弹出层宽度
			height : "400",	// 弹出层高度
			close : function() {
				console.log("右上角点击X关闭时的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	});
</script>

适用于登录窗口

<button id="demo3" class="javaex-btn primary">点击测试</button>

<script>
	$("#demo3").click(function() {
		javaex.dialog({
			type : "dialog",
			title : "添加",
			url : "http://www.javaex.cn/",	// 页面地址或网址或请求地址
			width : "600",	// 弹出层宽度
			height : "400",	// 弹出层高度
			close : function() {
				console.log("右上角点击X关闭时的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	});
</script>

参数说明

参数 默认值 说明 版本
id 随机ID 指定id，仅在当页面存在多个弹出层，需要关闭指定弹出层时才使用。配合parent.javaex.close(id);使用
type window或dialog
hasMask true 是否使用遮罩。如果你想无限弹窗，必须将hasMask设置为false
title 对话框标题
url 页面地址或网址或请求地址
content 如果url为空时，会填充content内容
width 800 弹出层的宽度（单位：像素）
height 弹出层的高度
scrolling yes 是否允许有滚动条。yes：有；no：没有
closeOnClickMask false 是否允许点击遮罩关闭弹窗
offsetLeft 0 偏移左边多少像素
offsetTop 0 偏移上边多少像素
close 右上角点击X关闭时的回调
opend 打开弹窗后的回调函数
opend : function() {
	console.log("弹窗已打开");
}
