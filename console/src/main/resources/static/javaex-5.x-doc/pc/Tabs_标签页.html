<script>
	javaex.tab({
		id : "tab",     // tab的id
		current : 1,    // 默认选中第几个选项卡，从1开始计
		mode : "mouseover",   // 鼠标移动切换选项卡
		callback : function (rtn) {
			// 切换选项卡后返回一个对象，包含选项卡的索引，从1开始计
			console.log(rtn.index);
		}
	});
</script>

参数说明

参数 默认值 说明 版本
id 选项卡id
current 1 默认显示第几个标签，从1开始计
mode 切换模式。mouseover：鼠标移动切换； click: 鼠标点击切换
isInit false 是否初始化就自动调用回调函数
callback 返回的对象中包含了选项卡的索引，从1开始计
display block div的display属性。常用的是block和flex，根据实际需要填写
delay 200 鼠标移动切换时，延迟多少毫秒
