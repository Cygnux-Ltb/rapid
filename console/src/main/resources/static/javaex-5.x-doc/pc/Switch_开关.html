Switch 开关

基础用法

<input type="checkbox" name="checkbox1" value="1" switch checked onchange="test(this)"/>

<script>
	function test(obj) {
		if ($(obj).is(":checked")) {
			console.log("选中了");
		} else {
			console.log("取消选中");
		}
	}
</script>

禁用状态

<input type="checkbox" name="checkbox2" value="2" switch checked disabled="disabled" onchange="test(this)"/>

尺寸 small

设置size="mini"，就是小尺寸

<input type="checkbox" name="checkbox3" value="3" switch size="small" checked onchange="test(this)"/>

支持文字

随便添加一个 class ，然后按照示例添加样式

<style>
	input[type="checkbox"][switch].switch-text:before {
		content: "√×";
	}
</style>

<input type="checkbox" class="switch-text" name="checkbox4" value="4" switch checked onchange="test(this)"/>
