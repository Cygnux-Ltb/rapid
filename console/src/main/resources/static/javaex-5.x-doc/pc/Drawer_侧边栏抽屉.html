Drawer 侧边栏抽屉

使用方法一：自定义页面

我是标题
// 这里编写你的代码 
<!-- 在页面上写一个如下的代码后再调用组件 -->
<div id="drawer" class="javaex-drawer">
	<div class="javaex-drawer-header">
		<div class="javaex-drawer-title">我是标题</div>
		<a class="javaex-drawer-close" href="javascript:;" onclick="hideDrawer()"><span class="icon-close"></span></a>
	</div>
	<div class="javaex-drawer-content">
		// 这里编写你的代码
	</div>
</div>

<button class="javaex-btn primary" onclick="test1()">点我弹窗</button>

<script>
	function test1() {
		javaex.drawer({
			id : "drawer",
			width : "500px",
			closeOnClickMask : true,
			position : "right",
			opend : function() {
				console.log("弹窗已打开");
			}
		});
	}

    /**
     * 隐藏侧边栏
     */
    function hideDrawer() {
        $(".javaex-mask").remove();
        $("#drawer").css("transform", "translateX(0px)");
    }
</script>

使用方法一：iframe加载另一个页面

<button class="javaex-btn primary" onclick="test2()">点我弹窗</button>

<script>
	function test2() {
		javaex.drawer({
			width : "500px",
			title : "我是标题",
			url : "https://www.baidu.com/",
			closeOnClickMask : true,
			position : "right",
			close : function() {
				console.log("点x关闭时产生的回调");
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			}
		});
	}
</script>

参数说明

参数 默认值 说明 版本
id 随机字符串 自定义页面必须指定id，iframe嵌套可以不指定
width 70% 弹出层宽度。支持： 500、500px、30% 这3种写法
position right 弹出位置，可选值：left、right
closeOnClickMask false 是否允许点击遮罩关闭
title 温馨提示 弹出层标题，自定义页面时不用填该选项
closeIcon true 是否显示右上角关闭 X，自定义页面时不用填该选项
url 使用iframe嵌套时需要该参数
close 点击右上角X关闭弹出层时的回调函数。仅在iframe嵌套时有效
opend 打开弹窗后的回调函数
opend : function() {
	console.log("弹窗已打开");
}
