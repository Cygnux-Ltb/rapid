关闭图标

<button id="demo2" class="javaex-btn default" block size="large">底部弹出</button>

<div id="popup2" class="javaex-popup javaex-popup-bottom" style="height:30%; display:none;"></div>

<script>
	$("#demo2").click(function() {
		javaex.popup({
			id : "popup2",
			position : "bottom",
			closeable : true,
			closeIconPosition : "top-right",
			close : function () {
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			},
			opend : function () {
				console.log("弹窗已打开");
			}
		});
	});
</script>

圆角弹框

添加 round 属性即可

<button id="demo6" class="javaex-btn default" block size="large">底部弹出</button>

<div id="popup2" class="javaex-popup javaex-popup-bottom" round style="height:30%; display:none;"></div>

<script>
	$("#demo2").click(function() {
		javaex.popup({
			id : "popup6",
			position : "bottom"
		});
	});
</script>

参数说明

参数 默认值 说明 版本
id 页面预设的 display:none; 的div的ID
position center 弹出位置（center,top,bottom,left,right）
mask black 遮罩颜色
black：黑色
transparent：透明
空字符串：无遮罩
closeOnClickMask true 是否点击遮罩关闭
closeable false 是否显示关闭按钮
closeIconPosition top-right 关闭按钮位置（top-left,top-right,bottom-left,bottom-right）
close 关闭弹框时触发的回调函数
opend : function() {
	console.log("弹窗已打开");
}
opend 打开弹框后触发的回调函数
opend : function() {
	console.log("弹窗已打开");
}
