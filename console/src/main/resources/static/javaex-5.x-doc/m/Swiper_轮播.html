参数说明

参数 默认值 说明 版本
id 容器ID
isAutoPlay true 是否自动轮播
focusCenter false 焦点是否自动居中
startSlide 1 默认显示第几张幻灯片，从1开始计
delay 3000 幻灯片自动轮播间隔时间，单位：毫秒
isInit false 是否初始化就自动调用回调函数
callback 返回的对象中包含了当前播放的幻灯片索引，从1开始计
