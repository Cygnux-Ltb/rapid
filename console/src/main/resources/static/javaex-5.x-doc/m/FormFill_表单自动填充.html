FormFill 表单自动填充

写在最前面

  • 文本框（input、hidden、textarea...）对应的值为空时，不做覆盖处理（如果页面上有后台渲染或写死的值的话）

标准写法

javaex.form({
	id : "form",    // 表单form的id
	formData : {
		"id" : data.id,
		"projectName" : data.projectName,
		"comment" : data.comment,
		"radio" : 0,           // 选项框的值允许为 null 或 空字符串
		"checkbox" : [1, 4]    // 复选框的值允许为字符串。例如： "1,2,3"，需要使用英文逗号，且不能有空格
	},
	callback : function() {
		// 渲染完成了，做点什么吧
	}
});

如果formData的属性一一对应，可以使用简化写法

javaex.form({
	id : "form",    // 表单form的id
	formData : data,
	callback : function() {
		// 渲染完成了，做点什么吧
	}
});

参数说明

参数 默认值 说明 版本
id 表单form的id。当然也不一定是<form>标签，只要规划一块区域范围的id即可
formData json数据
defaultRadioChecked 1 单选框没有值时，默认选中第几个（0代表不默认选中）
callback 数据填充完成后的回调函数
