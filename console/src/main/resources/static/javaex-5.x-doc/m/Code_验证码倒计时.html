Code 验证码倒计时

核心代码

<a href="javascript:;" style="color: #0087e0;" id="get-code" onclick="getCode()">获取验证码</a>

<script>
	function getCode() {
		javaex.code({
			id : "get-code"
		});
	}
</script>

参数说明

参数 默认值 说明 版本
id 点击触发的元素id
second 45 倒计时秒数
text 获取验证码 倒计时结束后，重新显示的文字
type 设为reset时，可以手动重置
