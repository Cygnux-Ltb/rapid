参数说明

参数 默认值 说明 版本
id 容器ID
alone false 是否只允许展开一个
isShowAll false 是否默认展开全部
open 点击展开后的回调函数，返回的对象中包含了当前点击的索引，从1开始计
opend : function(rtn) {
	console.log(rtn.index);
}
close 点击收起后的回调函数，返回的对象中包含了当前点击的索引，从1开始计
close : function(rtn) {
	console.log(rtn.index);
}
