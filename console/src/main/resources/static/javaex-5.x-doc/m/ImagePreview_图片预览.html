ImagePreview 图片预览

单图放大

<div class="javaex-image" style="width:100px;height:100px;">
	<img class="javaex-img" fit="cover" src="http://webimg.javaex.cn/Ft0-o8l5oCgH9dQUV67eMVODZVU_" onclick="img(this)"/>
</div>

<script>
	// obj为当前点击的元素对象
	function img(obj) {
		javaex.imagePreview({
			type : "image",	// 弹出层类型
			url : obj.src	// 图片地址
		});
	}
	
	// 或者下面这种写法
	$("img").click(function() {
		javaex.imagePreview({
			type : "image",	// 弹出层类型
			url : $(this).attr("src")	// 图片地址
		});
	});
</script>

参数说明

参数 默认值 说明 版本
type 必须是image
url 图片地址，就按照示例中的写
opend 打开弹窗后的回调函数
opend : function() {
	console.log("弹窗已打开");
}
