DatePicker 日期选择器（只能在移动端使用）

<div class="javaex-cell">
	<div class="javaex-cell-title javaex-field-label">请选择日期</div>
	<div class="javaex-cell-value javaex-field-value">
		<div class="javaex-field-body">
			<input id="date" type="text" class="javaex-field-control tr" placeholder="点击选择" autocomplete="off" readonly/>
		</div>
	</div>
</div>

<script>
	javaex.date({
		id : "date",
		title : "选择日期",
		time : "today",	// 默认选择的日期
		// 重新选择日期之后返回一个时间对象
		callback: function (rtn) {
			console.log(rtn.date);
		}
	});
</script>

参数说明

参数 默认值 说明 版本
id date容器的ID
title 不为空时，会显示标题
time 默认时间（格式：yyyy-MM-dd），也可以填写 today，表示当天日期
closeOnClickMask true 是否点击遮罩关闭
callback 返回的对象中包含了选中的值和选中的文本
