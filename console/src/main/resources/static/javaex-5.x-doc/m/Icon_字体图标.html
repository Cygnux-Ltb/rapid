Icon 字体图标

使用方法

直接通过设置类名为 icon-home2 来使用即可。例如：

<!--修改对应的class，可更换图标。不一定是 span 标签，也可以是 i 标签之类的-->
<span class="icon-home2"></span>

修改大小和颜色后的字体图标演示

<!--修改对应的class，可更换图标。不一定是 span 标签，也可以是 i 标签之类的-->
<span class="icon-home2" style="font-size:50px;color:red;"></span>
