Tag 标签

基础用法

支持 default、primary、success、warning 4种类型

<span class="javaex-tag javaex-tag-primary">标签</span>
<span class="javaex-tag javaex-tag-success">标签</span>
<span class="javaex-tag javaex-tag-danger">标签</span>
<span class="javaex-tag javaex-tag-warning">标签</span>

空心样式

通过 plain 属性将标签设置为空心

<span class="javaex-tag javaex-tag-primary" plain>标签</span>
<span class="javaex-tag javaex-tag-success" plain>标签</span>
<span class="javaex-tag javaex-tag-danger" plain>标签</span>
<span class="javaex-tag javaex-tag-warning" plain>标签</span>

圆角样式

通过 round 属性设置圆角

<span class="javaex-tag javaex-tag-primary" round>标签</span>

标签大小

通过 large 属性设置大标签

<span class="javaex-tag javaex-tag-primary" large>标签</span>
