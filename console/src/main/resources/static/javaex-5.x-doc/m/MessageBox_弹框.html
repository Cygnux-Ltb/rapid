MessageBox 弹框

确定弹框（不填 title 属性，则不带标题）

<button id="demo1" class="javaex-btn default" block size="large">确定弹框（带标题）</button>

<script>
	$("#demo1").click(function() {
		var id = "111";
		javaex.alert({
			title : "我是标题",
			content : "我是来测试的",
			confirm : function() {
				console.log("获取变量值：" + id);
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			},
			opend : function() {
				console.log("弹窗已打开");
			}
		});
	});
</script>

参数说明

参数 默认值 说明 版本
type 填写 button 时，为圆角按钮样式
title 标题（不填时不显示）
content 提示内容
confirmName 确定 确定按钮文字
confirm 点击确定按钮的回调
opend 打开弹窗后的回调函数
opend : function() {
			console.log("弹窗已打开");
		}

确定取消弹框

<button id="demo3" class="javaex-btn default" block size="large">确定取消弹框</button>

<script>
	$("#demo3").click(function() {
		var id = "333";
		javaex.confirm({
			content : "确定要删除么？",
			confirm : function() {
				console.log("获取变量值：" + id);
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			},
			cancel : function() {
				console.log("获取变量值：" + id);
				// 如果你想阻止弹出层关闭，直接 return false; 即可
				// return false;
			},
			opend : function() {
				console.log("弹窗已打开");
			}
		});
	});
</script>

参数说明

参数 默认值 说明 版本
type 填写 button 时，为圆角按钮样式
title 标题（不填时不显示）
content 提示内容
confirmName 确定 确定按钮文字
cancelName 取消 取消按钮文字
confirm 点击 确定 按钮的回调
cancel 点击 取消 按钮的回调
opend 打开弹窗后的回调函数
opend : function() {
			console.log("弹窗已打开");
		}
