Progress 线形进度条

基础用法

<div class="javaex-progress">
	<div id="progress1" class="javaex-progress-level"></div>
</div>

<script>
	javaex.progress({
		id : "progress1",
		percent : "40"
	});
</script>

百分比内显

百分比不占用额外控件，适用于文件上传等场景。

如果要显示百分比，则需要自己为 class="javaex-progress" 设定 line-height

<!--如果要显示百分比，则需要自己为 class="javaex-progress" 设定 line-height-->
<div class="javaex-progress" style="line-height: 16px;">
	<div id="progress2" class="javaex-progress-level"></div>
</div>

<script>
	javaex.progress({
		id : "progress2",
		percent : "40",
		isShowPercent : true
	});
</script>

自定义样式

<style>
.myclass{
	line-height: 16px;
}
.myclass .javaex-progress-level > span{
    background: linear-gradient(135deg,#fa2c19 0%,#fa6419 100%);
    border-radius: 5px;
    padding: 3px 5px 3px 6px;
}
</style>

<div class="javaex-progress myclass">
	<div id="progress3" class="javaex-progress-level"></div>
</div>

<script>
	javaex.progress({
		id : "progress3",
		percent : "40",
		isShowPercent : true
	});
</script>

参数说明

参数 默认值 说明 版本
id 指定操作哪个进度条
percent 0 进度条百分比，例如：40 或 "40"
isShowPercent false 是否显示百分比文字（如果要显示百分比，则需要自己为 class="progress" 设定 line-height）
