	javaex.tabSelect({
		id : "tab",
		current : 1,
		callback: function (rtn) {
			console.log(rtn.index);
		}
	});
</script>

参数说明

参数 默认值 说明 版本
id 选项卡的ID
current 1 默认选中第几个标签，从1开始计
isInit true 是否初始化就自动调用回调函数
callback 返回的对象中包含了选项卡的索引，从1开始计
