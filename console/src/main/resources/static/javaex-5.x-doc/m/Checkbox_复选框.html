Checkbox 复选框

基础用法

  • 默认点击文字也会勾选，可以不用在外面包一层label。
  • 如果没有文字，则必须写一个空格 或者 直接用label标签包起来。
  • 如果是ajax获取数据后拼接的代码，需要调用 javaex.render(); 重新渲染样式。
<input type="checkbox" fill name="checkbox" value="1"/>复选框未选中<br>
<input type="checkbox" fill name="checkbox" value="2" checked/>复选框选中<br>
<input type="checkbox" fill name="checkbox" value="3" disabled="disabled"/>复选框未选中（只读）<br>
<input type="checkbox" fill name="checkbox" value="4" checked disabled="disabled"/>复选框选中（只读）<br>

全选 | 取消全选

  • 添加 listen 属性。父级：listen-X，子级： listen-X-Y
  • X、Y并不仅限于数字，例如：父级：listen-table，子级： listen-table-next 也是可以的
<input type="checkbox" fill name="checkbox4" listen="listen-1" value="1"/>全选<br>
<input type="checkbox" fill name="checkbox4" listen="listen-1-2" value="2" />子级复选框<br>
<input type="checkbox" fill name="checkbox4" listen="listen-1-2" value="3" disabled="disabled"/>子级复选框（禁用）<br>
<input type="checkbox" fill name="checkbox4" listen="listen-1-2" value="4" />子级复选框<br>

针对复选框的 listen="listen-x" 属性全选。如果你想监听例如至少勾选一个复选框，才解除按钮的禁止点击状态，请调用如下方法，并将你的逻辑写在回调函数里

// 监听复选框选中事件
javaex.listenCheckbox({
	callback : function() {
		// 把你的逻辑写在这里
	}
});

伪单选框

一定要在外面包一层label，并且需要自己写一点点样式

<style>
	.javaex-btn.blue {
	    color: rgb(255, 255, 255);
	    background-color: rgb(64, 158, 255);
	}
	.javaex-btn.wathet {
	    color: rgb(16, 140, 238);
	    background-color: rgb(234, 246, 254);
	}
</style>

<label class="javaex-btn"><input type="checkbox" fake name="checkbox3" value="1" checked/>入门型</label>
<label class="javaex-btn"><input type="checkbox" fake name="checkbox3" value="2"/>标准型</label>
<label class="javaex-btn"><input type="checkbox" fake name="checkbox3" value="3"/>企业型</label>
<label class="javaex-btn"><input type="checkbox" fake name="checkbox3" value="4"/>豪华型</label>

<script>
	javaex.fakeCheckbox({
		name : "checkbox3",
		uncheckClass : "wathet", // 未选中时添加的class
		checkedClass : "blue" 	// 选中时添加的class
	});
</script>

参数说明

参数 默认值 说明 版本
name checkbox的name属性
uncheckClass 未选中时往label标签添加的class
checkedClass 选中时往label标签添加的class
