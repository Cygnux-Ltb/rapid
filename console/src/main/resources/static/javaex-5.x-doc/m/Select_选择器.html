Select 选择器（只能在移动端使用）

基础用法

<div class="javaex-cell">
	<div class="javaex-cell-title javaex-field-label">请选择城市</div>
	<div class="javaex-cell-value javaex-field-value">
		<div class="javaex-field-body">
			<select id="select1">
				<option value="">请选择</option>
				<option value="1">北京市</option>
				<option value="2" selected>天津市</option>
				<option value="3">沈阳市</option>
				<option value="4">长春市</option>
				<option value="5">哈尔滨市</option>
				<option value="6">上海市</option>
				<option value="7">南京市</option>
				<option value="8">武汉市</option>
				<option value="9">广州市</option>
				<option value="10">重庆市</option>
			</select>
		</div>
	</div>
</div>

<script>
	javaex.select({
		id : "select1",
		title : "请选择城市",
		callback: function (rtn) {
			console.log(rtn);
		}
	});
</script>

可搜索

<div class="javaex-cell">
	<div class="javaex-cell-title javaex-field-label">请选择城市</div>
	<div class="javaex-cell-value javaex-field-value">
		<div class="javaex-field-body">
			<select id="select2">
				<option value="">请选择</option>
				<option value="1">北京市</option>
				<option value="2" selected>天津市</option>
				<option value="3">沈阳市</option>
				<option value="4">长春市</option>
				<option value="5">哈尔滨市</option>
				<option value="6">上海市</option>
				<option value="7">南京市</option>
				<option value="8">武汉市</option>
				<option value="9">广州市</option>
				<option value="10">重庆市</option>
			</select>
		</div>
	</div>
</div>

<script>
	javaex.select({
		id : "select2",
		title : "请选择城市",
		searchable : true,
		callback: function (rtn) {
			console.log(rtn);
		}
	});
</script>

参数说明

参数 默认值 说明 版本
id select的ID
title 选择框的标题，为空时，不显示
closeOnClickMask true 是否点击遮罩关闭
searchable false 是否允许搜索，目前仅可搜索已加载的数据，不支持异步加载
searchPlaceholder 请输入搜索关键词 搜索框的提示内容
searchMode 1 搜索模式。1：即使搜索；2：按回车键搜索
callback 返回的对象中包含了选中的值和选中的文本
isInit false 是否初始化就自动调用回调函数 5.3
