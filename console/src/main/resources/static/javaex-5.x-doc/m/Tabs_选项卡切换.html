Tabs 选项卡切换

基础用法

<div id="tab1" class="javaex-tabs">
	<div class="javaex-tabs-wrap">
		<div class="javaex-tabs-nav">
			<div class="javaex-tab"><span class="javaex-tab-text">标签 1</span></div>
			<div class="javaex-tab"><span class="javaex-tab-text">标签 2</span></div>
			<div class="javaex-tab"><span class="javaex-tab-text">标签 3</span></div>
			<div class="javaex-tab"><span class="javaex-tab-text">标签 4</span></div>
		</div>
	</div>
	<div class="javaex-tabs-content">
		<div class="javaex-tab-panel">内容 1</div>
		<div class="javaex-tab-panel">内容 2</div>
		<div class="javaex-tab-panel">内容 3</div>
		<div class="javaex-tab-panel">内容 4</div>
	</div>
</div>

<script>
	javaex.tab({
		id : "tab1",
		current : 1,
		callback: function (rtn) {
			console.log(rtn.index);
		}
	});
</script>

切换动画

切换动画时，比基础用法多了一层 <div class="javaex-swipe-track"> 

<div id="tab2" class="javaex-tabs">
	<div class="javaex-tabs-wrap">
		<div class="javaex-tabs-nav">
			<div class="javaex-tab"><span class="javaex-tab-text">标签 1</span></div>
			<div class="javaex-tab"><span class="javaex-tab-text">标签 2</span></div>
			<div class="javaex-tab"><span class="javaex-tab-text">标签 3</span></div>
			<div class="javaex-tab"><span class="javaex-tab-text">标签 4</span></div>
		</div>
	</div>
	
	<div class="javaex-tabs-content">
		<!--使用切换动画时，需要包裹这一层-->
		<div class="javaex-swipe-track">
			<div class="javaex-tab-panel">内容 1</div>
			<div class="javaex-tab-panel">内容 2</div>
			<div class="javaex-tab-panel">内容 3</div>
			<div class="javaex-tab-panel">内容 4</div>
		</div>
	</div>
</div>

<script>
	javaex.tab({
		id : "tab2",
		mode : "slide",
		current : 1,
		callback: function (rtn) {
			console.log(rtn.index);
		}
	});
</script>

参数说明

参数 默认值 说明 版本
id 选项卡的ID
current 1 默认选中第几个标签，从1开始计
isInit false 是否初始化就自动调用回调函数
mode 内容切换模式，填写 slide 时，表示切换动画
callback 返回的对象中包含了选项卡的索引，从1开始计
