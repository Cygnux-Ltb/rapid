开始使用

首先，创建一个手机网页文档

<!DOCTYPE html PUBLIC "-//WAPFORUM//DTD XHTML Mobile 1.0//EN" "http://www.wapforum.org/DTD/xhtml-mobile10.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<!-- 防止苹果手机双击放大页面 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
<title>无标题文档</title>
</head>

<body>
</body>
</html>

然后，你需要加载以下文件

<!--字体图标-->
<link href="javaex-5.5.1/m/css/icomoon.css" rel="stylesheet" />
<!--骨架样式-->
<link href="javaex-5.5.1/m/css/common.css" rel="stylesheet" />
<!--皮肤-->
<link href="javaex-5.5.1/m/css/skin/default.css" rel="stylesheet" />
<!--jquery-->
<script src="javaex-5.5.1/m/lib/jquery-3.3.1.min.js"></script>
<!--移动端适配-->
<script src="javaex-5.5.1/m/js/flexible.js"></script>
<!--核心组件，不要压缩，压缩后部分功能可能会失效-->
<script src="javaex-5.5.1/m/js/javaex.js"></script>
<!--表单验证-->
<script src="javaex-5.5.1/m/js/javaex-formVerify.js"></script>

如果你只是想测试，可以加载远程文件进行测试
