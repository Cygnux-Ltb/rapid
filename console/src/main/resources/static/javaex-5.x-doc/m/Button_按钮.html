Button 按钮

按钮类型

按钮支持 default、primary、success、warning、danger、info 6种类型

<button class="javaex-btn default">默认按钮</button>
<button class="javaex-btn primary">主要按钮</button>
<button class="javaex-btn success">成功按钮</button>
<button class="javaex-btn warning">警告按钮</button>
<button class="javaex-btn danger">危险按钮</button>
<button class="javaex-btn info">信息按钮</button>

朴素按钮

通过 plain 属性将按钮设置为朴素按钮，朴素按钮的文字为按钮颜色，背景为白色（不支持default类型按钮）

<button class="javaex-btn primary" plain>朴素按钮</button>
<button class="javaex-btn success" plain>朴素按钮</button>

按钮形状

通过 square 设置方形按钮，通过 round 设置圆形按钮

<button class="javaex-btn primary" square>方形按钮</button>
<button class="javaex-btn primary" round>圆角按钮</button>

禁用状态

通过 disabled 属性来禁用按钮，禁用状态下按钮不可点击

<button class="javaex-btn primary" disabled>禁用状态</button>
<button class="javaex-btn success" round disabled>禁用状态</button>
<button class="javaex-btn danger" plain disabled>禁用状态</button>

图标按钮

<button class="javaex-btn danger" plain><span><i class="icon-star"></i></span></button>
<button class="javaex-btn danger"><span><i class="icon-star_border"></i> 收藏</span></button>

按钮尺寸

支持 large、normal、small、mini 四种尺寸，默认为 normal

<button class="javaex-btn primary" size="large">大号按钮</button>
<button class="javaex-btn primary">默认大小</button>
<button class="javaex-btn primary" size="small">小号按钮</button>
<button class="javaex-btn primary" size="mini">迷你按钮</button>

块级元素

按钮在默认情况下为行内块级元素，通过 block 属性可以将按钮的元素类型设置为块级元素，常用来实现通栏按钮

<button class="javaex-btn primary" block size="large">块级元素</button>
