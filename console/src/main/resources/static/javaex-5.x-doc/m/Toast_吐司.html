Toast 吐司

基础用法

<button id="demo1" class="javaex-btn default">加载提示（不会自动销毁）</button>
<button id="demo2" class="javaex-btn default">成功提示</button>
<button id="demo3" class="javaex-btn default">失败提示</button>

<script>
	// type=loading时，弹出层不会自动消失，用户也无法点击页面元素
	// 调用其他 javaex.toast(); 方法时会自动关闭加载层
	// 也可以调用 javaex.close(); 方法来手动关闭加载层
	$("#demo1").click(function() {
		javaex.toast({
			content : "加载中...",
			type : "loading"
		});
	});

	$("#demo2").click(function() {
		javaex.toast({
			content : "操作成功",
			type : "success"
		});
	});

	$("#demo3").click(function() {
		javaex.toast({
			content : "请求失败",
			type : "error"
		});
	});
</script>

参数说明

参数 默认值 说明 版本
type 弹出层类型。
loading：加载提示。success：成功提示。error：失败提示。
type=loading时，弹出层不会自动消失，用户也无法点击页面元素
content 消息内容
hasMask true 是否显示透明遮罩，显示后，用户无法点击页面元素
live 2000 提示存在时间，单位:毫秒 （对type=loading无效）
opend 打开后的回调函数
opend : function() {
	console.log("弹窗已打开");
}

自定义图标

<button id="demo4" class="javaex-btn default">自定义图标</button>

<script>
	// icon不仅限于本框架的字体图标，也可以使用其他字体图标库的
	$("#demo4").click(function() {
		javaex.toast({
			content : "自定义图标",
			type : "info",
			icon : "icon-volume_up"
		});
	});
</script>

参数说明

参数 默认值 说明 版本
type 必须填写 info
content 消息内容
hasMask true 是否显示透明遮罩，显示后，用户无法点击页面元素
icon 字体图标
live 2000 提示存在时间，单位:毫秒 （对type=loading无效）
opend 打开后的回调函数
opend : function() {
	console.log("弹窗已打开");
}

纯消息内容

<button id="demo5" class="javaex-btn default">纯消息内容</button>

<script>
	$("#demo5").click(function() {
		javaex.toast({
			content : "自定义消息内容",
			type : "message"
		});
	});
</script>

参数说明

参数 默认值 说明 版本
type 必须填写 message
content 消息内容
hasMask true 是否显示透明遮罩，显示后，用户无法点击页面元素
bottom 自定义距离底部的距离。
默认弹出层居中显示，可以填写 10%，20%... 或 100px，150px...
live 2000 提示存在时间，单位:毫秒 （对type=loading无效）
opend 打开后的回调函数
opend : function() {
	console.log("弹窗已打开");
}
