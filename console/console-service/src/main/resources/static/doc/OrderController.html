5.3. 创建订单 [前端调用: 开仓, 平仓, 一键平仓]

URL: /order

Type: POST

Content-Type: APPLICATION_JSON_UTF8

Description: 创建订单 [前端调用: 开仓, 平仓, 一键平仓]

Query-parameters:

Parameter Type Description Required Since

userId

int32

用户ID

false

-

instrumentCode

string

交易标的 (例: 期货代码, 股票代码)

false

-

side

int32

订单交易类型 (1:买入, 2:卖出)

false

-

action

int32

交易动作 (1: 开仓, 2: 平仓)

false

-

offerPrice

double

委托价格

false

-

offerQty

int32

委托数量

false

-

type

int32

订单类型 (1:COMMON, 2:FAK, 3:FOK) [可为空]

false

-

valid

int32

有效类型 (1:GTC, 2:GTD, 3:GFD) [可为空]

false

-

Request-example:

curl -X POST -H 'Content-Type: APPLICATION_JSON_UTF8' -i /order --data 'userId=0&side=0&action=0&offerPrice=0.0&offerQty=0&type=0&valid=0&instrumentCode='

Response-example:

OK
