5.3. 创建订单 [前端调用: 开仓, 平仓, 一键平仓]

URL: /order

Type: POST

Content-Type: APPLICATION_JSON_UTF8

Description: 创建订单 [前端调用: 开仓, 平仓, 一键平仓]

Query-parameters:

Parameter Type Description Required Since

userId

int32

[不可为空]用户ID

false

-

instrumentCode

string

[不可为空]交易标的 (例: 期货代码, 股票代码)

false

-

strategyName

string

[可为空]策略名

false

-

side

int32

[不可为空]订单交易类型 (1:买入, 2:卖出)

false

-

action

int32

[不可为空]交易动作 (1:开仓; 2:平仓)

false

-

offerPrice

double

[不可为空]委托价格

false

-

priceLimitType

int32

[可为空, 默认: 0]价格限制类型 (0:无限制; 1:涨停不卖; 2:跌停不买; 3:涨跌停同时限制)

false

-

offerQty

int32

[不可为空]委托数量

false

-

offerStartTime

string

[可为空]委托开始时间, 可精确到毫米, 时间格式[yyyymmdd-HHMMSS.SSS]

false

-

offerEndTime

string

[可为空]委托结束时间, 可精确到毫米, 时间格式[yyyymmdd-HHMMSS.SSS]

false

-

type

int32

[可为空, 默认: 1]订单类型 (1:COMMON, 2:FAK, 4:FOK)

false

-

valid

int32

[可为空, 默认: 1]有效类型 (1:GTC, 2:GTD, 3:GFD)

false

-

requestTime

int64

[可为空]请求时间, 前端请求时间戳, 可自行填充

false

-

Request-example:

curl -X POST -H 'Content-Type: APPLICATION_JSON_UTF8' -i /order --data 'userId=0&side=0&action=0&offerPrice=0.0&priceLimitType=0&offerQty=0&type=0&valid=0&requestTime=0&offerStartTime=&instrumentCode=&offerEndTime=&strategyName='

Response-example:

OK
